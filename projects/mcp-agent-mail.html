MCP Agent Mail

Gmail for your coding agents

When you run multiple AI coding agents in parallel — one on the backend, one on the frontend, one on infra — they step on each other. They overwrite edits, miss context from parallel workstreams, and require humans to relay messages between them.

MCP Agent Mail is a coordination layer: an inbox/outbox system with memorable agent identities, searchable message history, and file reservation leases so agents don't conflict. Backed by Git (human-auditable) and SQLite (indexed).

Think of it as async email + directory + change-intent signaling for your agent swarm.
