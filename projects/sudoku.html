Sudoku Solver

JavaScript · Web app

A web-based Sudoku solver that works like a human — using constraint propagation before falling back to backtracking. Shows each logical step: single options, naked and hidden pairs, line elimination.

Built as a web app so you can input any puzzle and watch it solve step by step.
